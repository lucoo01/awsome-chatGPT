### ChatGPT 插件
#### [ChatGPT Extension for VSCode](https://github.com/mpociot/chatgpt-vscode)
Visual Studio Code 扩展，允许你使用非官方的 ChatGPT API 来生成 OpenAI 的 ChatGPT 对话。
提出问题或使用编辑器中的代码片段，通过侧边栏的输入框查询 ChatGPT
在代码选择上点击右键，运行上下文菜单中的一个快捷方式

对回答提出后续问题
